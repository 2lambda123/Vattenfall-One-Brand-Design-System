The topbar contains a centered logotype and can be built as a starting point for a header.

```html
responsive: true
showSource: true
---
<header class="vf-topbar">
  <div class="vf-container">
    <a href="#" class="vf-topbar__logo">
      <img src="/img/vf-logo.svg" />
    </a>
  </div>
</header>
```
