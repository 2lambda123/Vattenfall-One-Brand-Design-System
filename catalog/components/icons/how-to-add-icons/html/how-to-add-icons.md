Several icons are available as fonts. More options will be added continuously.

# How to add an icon

Simply add one of the below classes to the element:

* `vf-icon-placeholder`
* `vf-icon-settings`
* `vf-icon-up`
* `vf-icon-down`
* `vf-icon-left`
* `vf-icon-right`
* `vf-icon-question`
* `vf-icon-map-tag`
* `vf-icon-time`
