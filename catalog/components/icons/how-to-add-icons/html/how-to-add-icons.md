Several icons are availble as fonts. This list will be continually expanded upon.

# How to add an icon

Simply add one of the below classes to the element:

* `vf-icon-placeholder`
* `vf-icon-settings`
* `vf-icon-up`
* `vf-icon-down`
* `vf-icon-left`
* `vf-icon-right`
* `vf-icon-question`
* `vf-icon-map-tag`
* `vf-icon-time`
