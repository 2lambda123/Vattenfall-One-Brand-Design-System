### Responsive utility classes

Available classes:
```
.vf-u--sm-hidden  - element hidden on small screen sizes and smaller
.vf-u--md-hidden  - element hidden on medium screen sizes and smaller
.vf-u--lg-hidden  - element hidden on large screen sizes and smaller
.vf-u--xl-hidden  - element hidden on extra-large screen sizes and smaller
```



```html
responsive: true
---
<div class="vf-u--sm-hidden  ">This element is hidden on small screen sizes and smaller</div>
<div class="vf-u--md-hidden  ">This element is hidden on medium screen sizes and smaller</div>
<div class="vf-u--lg-hidden  ">This element is hidden on large screen sizes and smaller</div>
<div class="vf-u--xl-hidden  ">This element is hidden on extra-large screen sizes and smaller</div>
```
